# 已学技能：神仙道3自动截图启动

### 推荐步骤顺序
- 在「快速引导」页点击「写脚本(需ROOT)」(430,1503) → 进入「我的腳本」
- 点击「未分类」(263,1242) → 展开脚本列表
- 用 OCR 确认目标脚本名「神仙道3自动截图」(358,777) 精确匹配（置信度100%）
- 点击脚本名 → 进入配置页（功能设置/循环设置/使用说明）
- 点击「加载」(575,1810) → 脚本加载（可能短暂跳转他页）
- 点击「启动」(540,1455) → 出现「开始运行」(538,1710) 提示
- 画面自动切换至目标游戏，脚本正常运行

### 关键 OCR 词
- 快速引导、写脚本(需ROOT)、我的腳本、未分类、神仙道3自动截图、加载、启动、开始运行

### 等待/广告处理
- 首次进入「快速引导」若下方有广告/推荐，无需处理，直接找到“写脚本”入口
- 点击「加载」后等待 <1s，若短暂跳转回桌面（游戏中心/广告页），等待稳定回来即可

### 易错点
- 必须精确匹配脚本显示名「神仙道3自动截图」，不点相似名称
- 必须先展开「未分类」分类，再点击脚本名；不能直接点空白处
- 点击「加载」后需等待页面出现「启动」按钮，再点击，不可连点

### 补充/差异（对照 SKILL.md）
- 标准流程只到“加载”或“运行”按钮，此脚本配置页多一道「启动」确认步骤（加载后需再点启动才能运行）
- 脚本列表含多个示例脚本（UI界面例子、演示例子），启动前务必二次OCR确认所选无误
- 需要ROOT权限才能进入「写脚本」入口，本设备已通过SU授权
