# 已学技能：爱琳诗篇启动

### 推荐步骤顺序
- 按键精灵「快速引导」页 → 点击「写脚本(需ROOT)」(430,1575)
- 进入「我的腳本」 → 点击「未分类」(263,1314) 展开
- 确认脚本名「爱琳诗篇-免费版自动截图」(438,414) 精确匹配后点击
- 进入配置页 → 点击「加载」(576,1810)
- 等待页面出现「启动」(540,1455) → 点击启动
- **立即调用 `wait_for_game_after_script_launch`**（90s超时）

### 关键 OCR 词
- 快速引导、写脚本(需ROOT)、我的腳本、未分类、1个脚本、爱琳诗篇-免费版自动截图、加载、启动

### 等待/广告处理
- 点击「加载」后短暂出现「加载中.」文字，页面随后变为「正在运行」状态，出现「启动」按钮
- 点击「启动」后屏幕切换至桌面/游戏中心界面（显示系统应用、小7手游等图标），帧率显示 `GT[HK]: 0ms S:0` 说明游戏开始启动
- 不要手动点击任何弹窗，由 `wait_for_game_after_script_launch` 自动检测游戏进程

### 易错点
- 必须精确匹配脚本显示名「爱琳诗篇-免费版自动截图」再点击，不点「分享脚本」「2MB」等相邻文字
- 必须先展开「未分类」分类，再点击脚本名；不能直接点「1个脚本」文字
- 点击「启动」后必须立即调用 `wait_for_game_after_script_launch`，禁止等待或手动点击隐私协议
- 不要在已检测到游戏进程后继续 tap/swipe

### 补充/差异（对照 SKILL.md）
- 本脚本「加载」后直接出现「启动」按钮，无「开始运行」中间提示，与标准流程有差异
- 点击「加载」后页面布局有短暂过渡（配置页→加载中→正在运行页面），坐标稳定后点击启动
- 点击「启动」后界面切换为桌面/游戏中心（非游戏中页），仍属于正常启动流程
- 本设备已授予ROOT权限，可直接使用「写脚本(需ROOT）」入口，无需处理权限弹窗
