# 已学技能：爱琳诗篇自动截图启动

### 推荐步骤顺序
- 按键精灵「快速引导」页 → 点击「写脚本(需ROOT)」（底部偏右）
- 进入「我的腳本」→ 点击「未分类」展开列表
- 确认脚本名「爱琳诗篇-免费版自动截图」精确匹配后点击
- 进入配置页 → 点击「加载」（底部居中）
- 观察短暂「加载中。」提示，等待自动过渡至新页面
- 新页面出现「启动」按钮，点击
- **立即**调用 `wait_for_game_after_script_launch`（90s超时）

### 关键 OCR 词
- 快速引导、写脚本(需ROOT)、我的腳本、未分类、1个脚本
- 爱琳诗篇-免费版自动截图、分享脚本、2MB
- 加载、加载中。、启动

### 等待/广告处理
- 点击「加载」后约0.5~1秒出现「加载中。」，之后页面自动变为「启动」面板，无需任何操作
- 页面底部可能出现「预下载开启，全图鉴直接送」等广告横幅（不可点击），忽略即可
- 点击「启动」后页面跳转至桌面/游戏中心（底部导航出现「·爱琳诗篇」标签），属于正常启动流程
- 使用 `wait_for_game_after_script_launch` 等待游戏进程（本设备第1次轮询即检测到）

### 易错点
- 必须先点击「未分类」文字展开脚本列表，不能直接点「1个脚本」或空白区域
- 点击脚本名时精确匹配「爱琳诗篇-免费版自动截图」，避免误触相邻的「分享脚本」「2MB」
- 加载前页面坐标与加载后不同：加载后页面环境/账号设置仍保留，但底部按钮变为「启动」而非「加载」
- 点击「启动」后不要手动等待弹窗，应**立即**调用游戏检测工具，系统会自动切换观察者
- 若设备未ROOT，入口可能为「制作脚本(免ROOT)」，需先确认是否有ROOT权限

### 补充/差异（对照 SKILL.md）
- 标准流程中「加载」后期望出现「开始运行」提示并保持原页面，但本脚本加载后直接跳转为新页面（含启动按钮），无独立提示
- 点击「启动」后页面直接跳转至桌面/游戏中心（非游戏内界面），属于按键精灵脚本的正常启动行为，无需额外关闭弹窗
- 本设备已ROOT，使用「写脚本(需ROOT)」入口；无ROOT设备需改走「制作脚本(免ROOT)」进入类似列表（流程其余部分相同）
