# 已学技能：爱琳诗篇启动

### 推荐步骤顺序
- 在「快速引导」页点击「写脚本(需ROOT)」(430,1575) → 进入「我的腳本」
- 点击「未分类」(263,1314) → 展开脚本列表，确认「1个脚本」
- 用 OCR 精确定位目标脚本名「爱琳诗篇-免费版自动截图」(438,414)，确认匹配
- 点击脚本名 → 进入配置页（功能设置/循环设置/使用说明）
- 点击「加载」(576,1810) → 脚本加载后页面变化，出现「启动」
- 点击「启动」(540,1455) → 脚本运行，自动拉起游戏
- **立即调用 `wait_for_game_after_script_launch`**，不可手动点击隐私协议

### 关键 OCR 词
- 快速引导、写脚本(需ROOT)、我的腳本、未分类、爱琳诗篇-免费版自动截图、加载、启动、用户协议和隐私政策、同意

### 等待/广告处理
- 点击「加载」后短暂跳转（可能回到快速引导），稳定后出现「启动」按钮，无需额外等待
- 点击「启动」后可能闪过小7手游等中间页，最终弹出「用户协议和隐私政策」，此时 **不要点击同意**，由 wait_for_game 自动检测游戏进程后切换为观察者

### 易错点
- 必须精确匹配脚本显示名「爱琳诗篇-免费版自动截图」，不点相似名称或「分享脚本」等文字
- 必须先展开「未分类」分类，再点击脚本名；不能直接点空白处或「1个脚本」文字
- 点击「启动」后必须立即调用 `wait_for_game_after_script_launch`，禁止等待手动点击隐私协议「同意」

### 补充/差异（对照 SKILL.md）
- 标准流程中「加载」后需等待「开始运行」提示，但本脚本配置页直接出现「启动」按钮，没有中间弹窗
- 启动后游戏立即弹出隐私协议而非直接进入游戏，但观察者会自动处理同意流程，执行者无需干预
- 本设备已授予 ROOT 权限，可直接进入「写脚本(需ROOT)」入口
