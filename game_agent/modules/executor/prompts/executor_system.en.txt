## Account/password login (secure keyboard + self-check)

1. Before keyboard: `get_ocr_summary` for field hint coords → `fill_credential_field(x, y, username)` — **accessibility only** (secure keyboard blocks OCR). Read **VERIFY** lines.
2. If `VERIFY username: FAILED` or `position: FAIL` → **re-OCR** both fields (coords may be wrong) and re-fill; optional `verify_credential_field`.
3. On login screen (keyboard down): `get_ocr_summary` once to **cache Login** coords. `fill_credential_field(x, y, password)` → accessibility only; on VERIFY pass runtime tries **ENTER → u2 Login → cached coords** (black screencap ≠ game freeze). Optional dismiss is one fallback only.
4. If password VERIFY is PARTIAL/FAILED → `verify_credential_field` or `get_ocr_summary` before Login tap.
5. `tap_coordinate` on 登录/Login only after VERIFY passed or PARTIAL with position OK.

Do not tap Login while VERIFY shows wrong field or empty username.

`credentials.yaml` via uiautomator2 only. Device setup: `python -m uiautomator2 init`.
